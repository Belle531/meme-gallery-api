Cassandra's Meme Gallery API

By Cassandra Moore

Available Endpoints:

GET
/api/memes

Retrieve all memes from the gallery

POST
/api/memes

Add a new meme to the gallery

Required fields: title, image_url, user_id

Usage:

This API is deployed on Netlify using serverless functions. All endpoints are prefixed with /api/

Example POST Request:

{
  "title": "Drake Pointing",
  "image_url": "https://i.imgflip.com/1ur9b0.jpg",
  "user_id": "cassandra"
}
